# Mongoose Schema

`DryerJS` generates `mongoose` schema from class definition.

## Customization

You can override the default behavior by using `@Schema` decorator.

```ts copy
@Schema({ name: 'tag' })
@Definition()
class Tag {}
```

You can also use `onSchema` option on `@DryerModule.register`.

```ts copy
// app.module.ts
@DryerModule.register({
  definitions: [Tag],
  onSchema: (schema, definition) => {
    schema.set('timestamps', true);
  },
});
```

## Index decorator



```ts copy
@Index({ name: 1 }, { unique: true })
@Definition()
class Tag {}
```

<Callout>
  `Index` is imported from `'dryerjs'`.
</Callout>
